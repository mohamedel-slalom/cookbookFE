# CookbookFE

React + Vite frontend starter for your Cookbook app.

## What is included

- Vite + React boilerplate
- Recipe Feed landing page
- Netflix-style horizontal recipe card carousel with left/right controls
- Ready structure to connect to your Spring Boot backend

## Run locally

1. `cd CookbookFE`
2. `npm install`
3. `npm run dev`

Optional production build:

4. `npm run build`

## Backend integration (next step)

When you are ready, we can replace mock recipe data in `src/App.jsx` with API calls to your Spring Boot service (for example `/api/recipes`), and configure `VITE_API_BASE_URL` for environments.
